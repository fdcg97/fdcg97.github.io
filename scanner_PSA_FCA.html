Escáner DIAGBOX PP2000

Este escáner es ideal para vehículos del grupo PSA. Proporciona diagnósticos precisos y detallados para una amplia gama de modelos. Su uso es sencillo y efectivo, garantizando un rendimiento óptimo para profesionales y aficionados.

  • Compatibilidad amplia con modelos PSA
  • Diagnósticos rápidos y precisos
  • Fácil de usar
  • Actualizaciones regulares de software

DIAGBOX PP2000 Interfaz de Diagnóstico para PSA

El DIAGBOX PP2000 es la interfaz de PC a vehículo compatible con múltiples aplicaciones de software, minimizando la inversión en interfaces de diagnóstico específicas de la marca. Funciona con DIAGBOX PP2000.

Características del DIAGBOX PP2000

  • Totalmente compatible con SAE-J2534-1 y SAE-J2534-2
  • Conectividad USB de vehículo a PC
  • Soporte para todos y la mayoría de los protocolos de vehículos actuales:K Line (ISO9141-2),High-Speed CAN (ISO 11898-2),Low-Speed CAN (ISO 11898-3),J1850 (SAE J1850),UDS Protocol.PWM Protocol. etc.
  • Funciona con Windows® XP
  • Conformidad de hardware con CE y FCC
  • Firmware actualizable

Detalles Específicos

  • Compatible con aplicaciones que diagnostican motores, transmisiones, ABS, paneles de instrumentos y más
  • Capacidades de flotas mixtas con amplio soporte para el protocolo
  • Acceso rápido a los datos del vehículo para informes de mantenimiento o gestión
  • Se adapta rápidamente al conector de diagnóstico del vehículo
  • Se puede utilizar con PC de escritorio o portátil
  • El software se actualiza a través del CD, el hardware se actualiza en línea
  • Compatible con DIAGBOX PP2000

Nota

  • DIAGBOX PP2000 solo se puede utilizar en un ordenador con Windows 7/8 de 64 bits.

Software Compatible

  • Compatible con diagnóstico basado en PC, Sistema de Diagnóstico Global DIAGBOX PP2000 y Sistema de Programación de Servicio en Línea (SPS)
  • Nueva aplicación de diagnóstico de software necesaria para todos los vehículos de Arquitectura Global PSA (Global A).
  • DIAGBOX PP2000 es un software que se ejecuta en un PC específico. Las actualizaciones se reciben en linea.

DIAGBOX PP2000 Proporciona las Siguientes Capacidades en Múltiples Idiomas

  • Lee los códigos de estado, datos de congelación (Freeze Frame) y borra los códigos
  • Lee el VIN, número de la ECU y número del software
  • Registra, almacena y reproduce los datos almacenados
  • Controla y muestra las funciones de control de salidas
  • Configura y realiza funciones de reseteo
  • Graba o muestra el historial de las sesiones de diagnóstico anteriores del vehículo

Sistema de Programación de Servicio (SPS)

  • Programación a través de paso en los modelos PSA antiguos y vehículos Global PSA
  • Aumenta la velocidad de reprogramación en un 20-70% y se prepara para cubrir los nuevos modelos
  • Para la programación a través de paso de las unidades de control electrónico en todos los vehículos fabricados desde 1996 y los futuros modelos
